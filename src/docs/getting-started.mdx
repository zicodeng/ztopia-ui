# Getting Started

To use Ztopia UI in your project, just install `ztopia-ui` as `devDependencies`.

    npm i -D ztopia-ui
    or
    yarn add --dev ztopia-ui

Import components and enjoy Ztopia UI.

```javascript
import React from 'react';
import ReactDOM from 'react-dom';

import { Button } from 'ztopia-ui';

class App extends React.Component {
    render() {
        return (
            <div>
                <Button>Click Me</Button>
            </div>
        );
    }
}

ReactDOM.render(<App />, document.getElementById('root'));
```
