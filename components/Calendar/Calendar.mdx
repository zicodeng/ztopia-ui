# Calendar

Dependencies: [react-big-calender](https://github.com/intljusticemission/react-big-calendar) and [moment](https://github.com/moment/moment)

## Demo

### Select Event

<Playground>
  <CalendarDemo>
    {({ onSelectEvent }) => (
      <Calendar
        onSelectEvent={onSelectEvent}
        events={[
          {
            allDay: true,
            title: 'Make the World Better Place',
            start: subDays(new Date(), 2),
            end: addDays(new Date(), 2),
          },
          {
            allDay: true,
            title: 'Camping',
            start: subDays(new Date(), 1),
            end: addDays(new Date(), 1),
          },
          {
            allDay: false,
            title: 'Study',
            start: setHours(setMinutes(new Date(), 0), 6),
            end: setHours(setMinutes(new Date(), 30), 16),
          },
          {
            allDay: false,
            title: 'Practice Guitar',
            start: setHours(setMinutes(new Date(), 0), 12),
            end: setHours(setMinutes(new Date(), 60), 14),
          },
        ]}
      />
    )}
  </CalendarDemo>
</Playground>

## Create Event

Click or drag to create a event

<Playground>
  <CalendarDemo>
    {({ events, onSelectSlot }) => (
      <Calendar isSelectable events={events} onSelectSlot={onSelectSlot} />
    )}
  </CalendarDemo>
</Playground>

## API

<Props of={Calendar} />

```ts
interface Event {
  allDay?: boolean;
  title?: string;
  start?: Date;
  end?: Date;
  resource?: any;
}

interface Slot {
  start: string | Date;
  end: string | Date;
  slots: Date[] | string[];
  action: 'select' | 'click' | 'doubleClick';
}
```
