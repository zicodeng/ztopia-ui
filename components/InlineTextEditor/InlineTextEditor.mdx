# InlineTextEditor

Dependencies: [medium-editor](https://github.com/yabwe/medium-editor) and [fontawesome](https://fontawesome.com/)

## Demo

### Basic Usage

Once you obtain a reference to editor instance through `onReady`, you can call `editor.getContent()` to manually access content

<Playground>
  <InlineTextEditorDemo>
    {({ onReady, printEditorContent }) => (
      <>
        <InlineTextEditor defaultValue={DEFAULT_VALUE} onReady={onReady} />
        <Button style={{ marginTop: 50 }} onClick={printEditorContent}>Print Editor Content</Button>
      </>
    )}
  </InlineTextEditorDemo>
</Playground>
