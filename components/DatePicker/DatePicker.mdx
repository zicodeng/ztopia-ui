# DatePicker

Dependencies: [react-datepicker](https://github.com/Hacker0x01/react-datepicker) and [date-fns](https://github.com/date-fns/date-fns)

## Demo

### Custom Input

<Playground>
  <Container style={{ height: 400, display: 'block' }}>
    <DatePickerDemo>
      {({ value, onChange }) => (
        <DatePicker
          input={<Input label="Birthday" variant="material" />}
          value={value}
          onChange={onChange}
        />
      )}
    </DatePickerDemo>
  </Container>
</Playground>

### Min Date, Max Date, Min Time & Max Time

Date or time out of the range will be disabled

<Playground>
  <Container style={{ height: 400, display: 'block' }}>
    <DatePickerDemo>
      {({ value, onChange }) => (
        <DatePicker
          isTimeSelectEnabled
          minTime={setHours(setMinutes(new Date(), 0), 10)}
          maxTime={setHours(setMinutes(new Date(), 30), 20)}
          minDate={subDays(new Date(), 5)}
          maxDate={addDays(new Date(), 5)}
          input={<Input label="Birthday" variant="material" />}
          value={value}
          onChange={onChange}
        />
      )}
    </DatePickerDemo>
  </Container>
</Playground>

### Highlight Dates

<Playground>
  <Container style={{ height: 400, display: 'block' }}>
    <DatePickerDemo>
      {({ value, onChange }) => (
        <DatePicker
          highlightDates={[subDays(new Date(), 7), addDays(new Date(), 7)]}
          input={<Input label="Birthday" variant="material" />}
          value={value}
          onChange={onChange}
        />
      )}
    </DatePickerDemo>
  </Container>
</Playground>
