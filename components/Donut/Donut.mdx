# Donut

## Demo

### Basic Usage

<Playground>
  <Donut
    nameAccessor="country"
    valueAccessor="population"
    data={Array.from({ length: 5 }).map((_, i) => ({
      country: faker.address.country(),
      population: Math.floor(Math.random() * 100000 + 10000),
    }))}
  />
</Playground>
